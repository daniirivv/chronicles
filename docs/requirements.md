# Requirements

### EPIC-01 Gestión de Libros

RU-01: Como usuario, quiero crear una entrada de libro con los siguientes atributos:
* Título
* Autor
* Año de publicación
* Formato
* Páginas
* Prioridad
* Galardones
* Valoración
* Observaciones
* Fecha de alta

para guardar los libros que me he leído y/o quiero leer en el futuro.

RU-02: Como usuario, quiero leer una entrada de libro para consultar los diferentes
campos con su respectiva información.

RU-03: Como usuario, quiero modificar una entrada de libro para actualizar su información
a lo largo del tiempo.

RU-04: Como usuario, quiero eliminar una entrada de libro para mantener mi historial actualizado
y relevante.
